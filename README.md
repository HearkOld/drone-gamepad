# Drone gamepad

Controlling drone via HTML5 gamepad API.

I'm using `Logitech Logitech Extreme 3D (Vendor: 046d Product: c215)"` for gamepad.

## Prerequisite

- node
- npm

## Develop

enter the command below

```
npm install
coffee app.coffee
```
